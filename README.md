# Telegram Contest Bot

Бот для проведения конкурсов в Telegram с автоматическим сбором заявок.

## Возможности

- Приём заявок от участников (минимум 3 ссылки на приглашённых)
- Валидация ссылок формата `t.me/username`
- Автоматическое сохранение в приватный канал (база данных)
- Защита от дубликатов
- Статистика участников
- Работает 24/7 на бесплатном хостинге

## Быстрый старт

1. Создай бота через @BotFather
2. Создай приватный канал и добавь бота как админа
3. Узнай ID канала через @userinfobot
4. Заполни переменные окружения на Render.com
5. Задеплой код

**Подробная инструкция:** [ИНСТРУКЦИЯ.md](ИНСТРУКЦИЯ.md)

## Структура проекта

```

├── bot.py              # Основной код бота

├── .env.example        # Пример настроек
├── render.yaml         # Конфигурация Render

```

## Переменные окружения

- `BOT_TOKEN` - токен бота от @BotFather
- `LOG_CHANNEL_ID` - ID приватного канала (с минусом, например: -1001234567890)

## Технологии

- Python 3.11+
- aiogram 3.15.0
- Render.com (бесплатный хостинг)

## Команды бота

- `/start` - Начать участие в конкурсе
- `/stats` - Показать статистику
